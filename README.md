# 🌊 OceanBreeze Resorts 🏖️

OceanBreeze Resorts is a **modern M-Commerce hotel room booking system** designed to provide a seamless experience for both customers and administrators.

* 📱 **Customer Mobile App** – Browse, book, and manage resort stays.
* 💻 **Admin Web Portal** – Manage rooms, pricing, availability, and check-outs.

---

## ✨ Features

### Customer App

* 🔐 Account Management (Sign up, login, edit profile)
* 🏨 Room Booking (type, quantity, location, AC/Non-AC)
* 💳 Secure Card Payments
* ❌ Cancel Bookings
* 📊 Real-Time Availability Updates

### Admin Web

* 🔐 Admin Authentication & Password Management
* 🏨 Room Management (update prices, adjust quantity)
* ✅ Customer Check-Out Handling
